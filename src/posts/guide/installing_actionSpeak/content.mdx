## 웹사이트 추가하기
1. actionSpeak에 로그인하여 [팝업 추가하러 가기] 또는 [대시보드]를 클릭합니다.

![](/posts/guide/installing_actionSpeak/1.png)

2. 웹사이트의 주소를 입력하고 [추가하기]를 클릭합니다.

![](/posts/guide/installing_actionSpeak/2.png)

3. 추가된 웹사이트를 확인합니다.

![](/posts/guide/installing_actionSpeak/3.png)


## 웹사이트에 설치하기 (아임웹 예시)

1. 추가된 웹사이트로 들어와서 [복사하기]를 클릭합니다.

![](/posts/guide/installing_actionSpeak/4.png)

2. 복사한 코드를  ‘사이트 관리 > 환경설정 > SEO(검색엔진최적화)’의 Header Code에 붙여넣습니다.

![](/posts/guide/installing_actionSpeak/5.png)


## 팝업 추가하기
> 🎯 고객이 웹사이트를 방문했을 때, 회원가입을 유도하는 팝업을 보여주겠습니다.

1. ‘팝업 관리’에서 기본 팝업을 선택합니다.

![](/posts/guide/installing_actionSpeak/6.png)

2. 미리보기를 확인하면서 설정 값을 입력합니다.
  - 버튼 링크: 버튼 클릭 시 이동할 페이지의 URL
  - 지속 시간: 팝업이 지속되는 시간
  - 제한 시간 표시: 팝업의 남은 시간을 보여줄지 여부
  - 빈도: 동일 고객에게 팝업이 노출되는 횟수
  - 대기 시간: 팝업이 나타나기 전 대기 시간

![](/posts/guide/installing_actionSpeak/7.png)

3. 저장 후 웹사이트에 들어가 팝업 노출을 확인합니다.

![](/posts/guide/installing_actionSpeak/8.png)
