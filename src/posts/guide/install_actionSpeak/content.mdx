## 웹사이트 추가하기
1. actionSpeak에 로그인하여 [팝업 추가하러 가기] 또는 [대시보드]를 클릭합니다.

![](/posts/guide/install_actionSpeak_guide/1.png)

2. 웹사이트의 주소를 입력하고 [추가하기]를 클릭합니다.

![](/posts/guide/install_actionSpeak_guide/2.png)

3. 추가된 웹사이트를 확인합니다.

![](/posts/guide/install_actionSpeak_guide/3.png)


## 웹사이트에 설치하기 (아임웹 예시)

1. 추가된 웹사이트로 들어와서 [복사하기]를 클릭합니다.

![](/posts/guide/install_actionSpeak_guide/4.png)

2. 복사한 코드를  ‘사이트 관리 > 환경설정 > SEO(검색엔진최적화)’의 Header Code에 붙여놓습니다.

![](/posts/guide/install_actionSpeak_guide/5.png)
